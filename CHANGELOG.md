# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Initial Release

### Added
- Mecha actor sheet with stats, dice, abilities, and items
- Enemy actor sheet with boss mode, attacks, and recharge attacks
- 12 item types (weapons, armor, modules, equipment, consumables, abilities, attacks)
- Roll-under d20 stat checks with advantage/disadvantage
- Reactor die degradation system
- Initiative system based on Mobility/System stat tests
- Macro support for all roll buttons
- Dark cyberpunk theme styling
- Chat message styling consistent with character sheet
- Restore HP/AP buttons with confirmation
- Enemy token HP auto-roll from hit dice
- Boss mode with boss attacks and recharge attacks
- Credits tracking for mecha actors
